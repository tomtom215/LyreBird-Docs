# Features

Complete feature list for LyreBirdAudio.

---

## Core Features

- **Automatic Hardware Detection** - Zero-configuration capability detection
- **USB Device Persistence** - Consistent device naming across reboots
- **Automatic Recovery** - Self-healing streams with watchdog monitoring
- **Production Ready** - Designed for 24/7 unattended operation
- **Easy Updates** - Git-based version management with rollback
- **Comprehensive Diagnostics** - Built-in health monitoring and reporting

---

## Audio Features

- Multiple codec support (Opus, AAC, MP3, PCM)
- Configurable sample rates and bit depths
- Bitrate optimization
- Multi-channel audio support
- Low-latency streaming

---

## Management Features

- Interactive orchestrator wizard
- Command-line management tools
- Systemd integration
- Automatic service start
- Comprehensive logging
